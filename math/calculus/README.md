# Calculus

This project covers summation and product notation, derivatives, partial
derivatives, indefinite and definite integrals, and double integrals.

## Python functions

- `summation_i_squared(n)` calculates the sum of the squares from 1 through
  `n` without using loops.
- `poly_derivative(poly)` calculates the derivative of a polynomial represented
  by a coefficient list.
- `poly_integral(poly, C=0)` calculates the integral of a polynomial represented
  by a coefficient list.
