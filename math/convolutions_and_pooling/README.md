# Convolutions and Pooling

This project implements convolution and pooling operations with NumPy.

## Files

- `0-convolve_grayscale_valid.py`: valid grayscale convolution
- `1-convolve_grayscale_same.py`: same grayscale convolution
- `2-convolve_grayscale_padding.py`: custom-padded grayscale convolution
- `3-convolve_grayscale.py`: strided grayscale convolution
- `4-convolve_channels.py`: convolution across image channels
- `5-convolve.py`: convolution using multiple kernels
- `6-pool.py`: max and average pooling
