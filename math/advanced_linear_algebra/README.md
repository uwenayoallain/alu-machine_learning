# Advanced Linear Algebra

This project implements determinants, minors, cofactors, adjugates, matrix
inverses, and definiteness classification from their mathematical definitions.

## Files

- `0-determinant.py`: calculates the determinant of a square matrix
- `1-minor.py`: calculates a matrix of minors
- `2-cofactor.py`: calculates a cofactor matrix
- `3-adjugate.py`: calculates an adjugate matrix
- `4-inverse.py`: calculates a matrix inverse
- `5-definiteness.py`: classifies the definiteness of a symmetric matrix
