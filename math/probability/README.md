# Probability

This project implements common probability distributions without importing
external modules. Each class can be initialized with distribution parameters
or estimate those parameters from a supplied data set.

## Distributions

- `Poisson`: probability mass and cumulative distribution functions
- `Exponential`: probability density and cumulative distribution functions
- `Normal`: z-score conversion, probability density, and cumulative functions
- `Binomial`: probability mass and cumulative distribution functions
