# Plotting

This project creates line, scatter, histogram, and stacked-bar plots with
Matplotlib, including a combined multi-plot figure.

## Files

- `0-line.py`: cubic line graph
- `1-scatter.py`: height and weight scatter plot
- `2-change_scale.py`: logarithmic carbon-14 decay plot
- `3-two.py`: comparison of two radioactive decay curves
- `4-frequency.py`: histogram of student grades
- `5-all_in_one.py`: five plots arranged in one figure
- `6-bars.py`: stacked quantities of fruit
