# Linear Algebra

This project covers slicing, matrix shapes, transposition, element-wise
operations, concatenation, and matrix multiplication with Python and NumPy.

## Files

- `0-slice_me_up.py` through `8-ridin_bareback.py`: list-based operations
- `9-let_the_butcher_slice_it.py`: NumPy slicing
- `10-ill_use_my_scale.py` through `14-saddle_up.py`: NumPy operations
